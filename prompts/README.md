# Prompts + AI Workflows

This folder is my prompt library and “AI operating system” for marketing + analytics.

## Purpose
- Consistent strategy outputs (briefs, content frameworks, campaign plans)
- Re-usable analysis prompts (posts, cohorts, influencer ROI)
- Draft faster, review smarter

## Categories
- `/strategy/` positioning, ICP, messaging
- `/instagram/` content, caption formulas, reels, hashtags
- `/influencer/` outreach scripts, negotiation angles
- `/analytics/` metric definitions + analysis prompts

## How to use
1. Copy a prompt.
2. Paste into your AI tool.
3. Replace bracketed variables.
4. Log outcomes + what changed.

## Minimum quality checklist
- One clear goal
- One success metric
- One constraint (time/budget/brand tone)
