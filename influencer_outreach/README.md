# influencer_outreach

Goal: build a repeatable system for creator seeding + outreach.

## Recommended structure
- `/creators/` (sourcing, ICP criteria, shortlists)
- `/outreach/` (email/DM templates, follow-up schedule)
- `/results/` (responses, posts delivered, ROI notes)

## Core metrics
- Response rate
- Share/post rate (posts / seeds)
- Cost per post (incl. product cost + time)
- Lifetime value (repeat posts, retention)

## Outreach fields (CRM)
- Handle + platform
- Audience size + niche + location
- Contact method (email/DM)
- Status (new / contacted / follow-up / shipped / posted)
- Outcome notes

## Workflow
1. Define ICP + goals (what audience, what category)
2. Source creators (hashtags, competitors, marketplaces)
3. Contact: pitch, value prop, clear CTA
4. Follow-up: set cadence + templates
5. Measure results: link to performance analysis + dashboard
