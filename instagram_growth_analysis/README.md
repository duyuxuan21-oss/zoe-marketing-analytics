# instagram_growth_analysis

Goal: analyse Instagram growth + content performance to drive recommendations.

## Recommended structure
- `/data/` (exports, scraped JSON/CSV)
- `/notebooks/` (Python notebooks or analysis scripts)
- `/reports/` (dashboard screenshots + written summary)

## Core metrics
- Follower growth % (week-over-week)
- Engagement rate = (likes + comments + saves + shares) / impressions
- Reach, saves, shares, comments

## Workflow
1. Intake: what business KPI are we moving?
2. Baseline: current metrics + competitor benchmark
3. Experiment: what change did we ship (hook, CTA, format, time)?
4. Result: compare before/after with context
