# Dashboards (Marketing Analytics)

Goal: build dashboards that answer real questions fast. If a metric can't drive a decision, it doesn't belong.

## Deliverables
- Campaign performance dashboards (weekly)
- Instagram account health (monthly)
- Influencer seeding pipeline tracker

## Data sources (examples)
- Instagram insights exports
- TikTok/Meta ads exports
- CSV logs from creator outreach

## Core metrics
- Reach, saves, shares
- Engagement rate (post + account level)
- CPA / CAC proxy
- Influencer ROI (blended)

## Suggested structure
- `/data/` raw exports + a `README` on columns
- `/transform/` cleaning scripts
- `/dashboard/` workbook / config
